============
Installation
============

In a :ref:`composer-based TYPO3 installation <t3start:install>` you can install
the extension EXT:content_information via composer:

.. code-block:: bash

   composer require xima/content-information

Clear all caches
----------------

In the module :guilabel:`Admin Tools > Maintanance` you can conveniently clear
all caches by clicking the button :guilabel:`Flush cache`.
